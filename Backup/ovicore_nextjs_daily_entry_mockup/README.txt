# OviCore Daily Production Entry — Next.js-style mockup

This package contains:

1. `ovicore_daily_production_entry_mockup.html`
   - Open this directly in a browser.
   - It is a visual/interactive prototype of how the Daily Production Entry screen could work.

2. `DailyProductionEntryMockup.tsx`
   - A Next.js/React client component version.
   - Drop it into a Next.js app using Tailwind CSS.

Prototype features:
- OviCore-style sidebar and workflow header
- Daily production table
- Frozen shed/flock column
- Editable daily fields
- Live calculated closing birds
- Live Hen Day %
- Live FCR/dozen
- Row status: Saved, Review suggested, Missing
- Copy yesterday button
- Mobile/tablet card layout
